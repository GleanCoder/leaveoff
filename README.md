# LeaveOff

LeaveOff is a backend-driven leave management system designed to simulate real-world organizational workflows such as leave requests, approval chains, and balance tracking.

---

## ⚙️ Tech Stack

_Backend_

- Node.js
- Express
- PostgreSQL
- Prisma ORM
- JWT Authentication
